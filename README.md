# FirstName BunchOfNumbers Detector

### Installation

```bash
git clone
pip install tweepy
pip install dotenv
python final.py
```

Request api keys: https://developer.twitter.com/.\
Store keys in `.env`.

```env
CONSUMER_KEY = 'YOUR_KEY'
CONSUMER_KEY_SECRET = 'YOUR_KEY'
ACCESS_TOKEN = 'YOUR_KEY'
ACCESS_TOKEN_SECRET = 'YOUR_KEY'
```
